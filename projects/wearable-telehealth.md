# Wearable Telehealth Device (ESP32)

## Overview
Final Year Project focused on the design and validation of a wearable electronic system for physiological data acquisition and wireless communication.

## Objectives
- Design a low-power embedded hardware platform
- Integrate analogue biomedical sensors
- Enable BLE/Wi-Fi communication for data transmission
- Validate system performance through bench testing

## System Design
- ESP32 microcontroller
- Analogue sensor interfaces with signal conditioning
- Power regulation and battery management
- BLE and Wi-Fi connectivity

## Engineering Work Performed
- Circuit design and sensor integration
- Analogue signal conditioning and filtering
- Firmware-hardware integration (no firmware focus)
- Bench testing, calibration, and fault isolation
- Latency and system validation testing

## Tools & Technologies
- ESP32
- LTspice
- Multimeter, oscilloscope (academic)
- Python / C / MATLAB
- Technical documentation and validation reports

## Outcomes
- Functional prototype demonstrating reliable data acquisition
- Validated communication latency and sensor performance
- Documented design trade-offs and system limitations

## Key Learnings
- Embedded hardware design trade-offs
- Importance of testability and validation
- Power and signal integrity considerations
